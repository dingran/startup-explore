Datavisual is a web-based data visualization production software tool targeting those in the design world with limited programming skills but still have the need to make beautiful charts and graphs.
With a mission is to make data visual accessible to everyone, Datavisual's simple user interface aims to empower individuals and organizations to easily create and share beautiful visualizations. Perfect for designers, journalists, researchers and bloggers with limited programming skills who have a need to visualize their data in a pleasing and dynamic way.
Datavisual's platform is entirely web-based so everything is created using HTML, CSS, JavaScript, Jade, Mongo, Node and Backbone. They are currently using the D3 library to generate the visualizations.
